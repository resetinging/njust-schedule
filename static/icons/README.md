# TabBar 图标

本目录需要放置以下 8 个图标文件（PNG 格式，建议 81×81 px）：

- `schedule.png`       — 课表（未选中）
- `schedule-active.png`  — 课表（选中）
- `exam.png`           — 考试（未选中）
- `exam-active.png`      — 考试（选中）
- `eval.png`           — 评教（未选中）
- `eval-active.png`      — 评教（选中）
- `settings.png`       — 我的（未选中）
- `settings-active.png`  — 我的（选中）

推荐图标资源：
- https://www.iconfont.cn/ 搜索：日历、考试、评价、用户
- 或者使用微信官方图标库

在正式发布前，请用实际图标替换此目录。
